**Todo Application**<br/>
Todo application developed using JSP, Servlet, JDBC and MySQL Database.<br/>

**Following features**<br/>
Add todo<br/>
Edit todo<br/>
List Dodo<br/>
Delete Todo<br/>
Login<br/>
Signup<br/>
Responsive layout<br/>

**Technology stack**<br/>
Java 8+<br/>
JSP and Servlet<br/>
JDBC<br/>
MySQL database<br/>
Eclipse IDE<br/>
Tomcat server 8+<br/>
